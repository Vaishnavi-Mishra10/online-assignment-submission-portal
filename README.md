# Online Assignment Submission Portal 🎓

A PHP & MySQL based portal for students and teachers to manage assignments online.

### ✨ Key Features
- **Role-Based Login:** Separate dashboards for Admin, Teacher, and Student
- **Dark Mode Toggle:** Modern UI with light/dark theme 🌙
- **Assignment Management:** Upload, edit, download assignments
- **Student Management:** Admin can add/edit students, view student list
- **Profile System:** Users can update their profile
- **Responsive Design:** Built with Bootstrap 5, works on mobile too

### 🛠️ Tech Stack
**Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript  
**Backend:** PHP 8+  
**Database:** MySQL
